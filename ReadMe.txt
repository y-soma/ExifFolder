ExifFolder
===============================================================================

[ 概要 ]
  Exifタグが付いている画像ファイルをタグ別にフォルダ分けする

[使い方]
   1."タグ選択"でフォルダ分けするタグを選ぶ
   2.目的の画像ファイルをドラッグ＆ドロップ
   終了 (成功すれば同じ場所にフォルダを作成し、画像を移動する)

  ※"コピーして移動する"にチェックを入れた場合、元のファイルを残してフォルダ移動します。
  ※失敗した場合は何も起こりません
